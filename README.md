# MultiThreadedHttpServer

II.3506 Web Technologies - Lab 4 : HTTP Server

## Objective

Write a multi-threaded HTTP Server

## Deployment

* Maven
At the root the project :
```
mvn install
mvn clean
mvn compile
```
Then to launch the main class : 
```
cd target/classes/
java httpd.Main config
```
